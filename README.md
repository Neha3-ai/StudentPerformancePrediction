# Student Performance Prediction System

## Overview
This project predicts the academic performance of students using Machine Learning algorithms based on various academic and personal factors.

## Technologies Used
- Python
- Flask
- Machine Learning (Scikit-learn)
- HTML
- CSS
- JavaScript
- Chart.js

## Features
- Student performance prediction using ML
- Early warning system for students at risk
- Data visualization dashboard
- What-if analysis for performance improvement
- AI-based suggestions
- PDF report generation

## Dataset Attributes
- Roll Number
- Study Hours
- Sleep Hours
- Play Hours
- Attendance
- Previous Marks
- Assignments Completed
- Final Marks (Predicted)

## Project Structure

StudentPerformancePrediction
│
├── dataset
├── backend
├── models
├── website
└── reports

## How to Run the Project

1. Install Python
2. Install required libraries:

pip install flask pandas scikit-learn matplotlib seaborn reportlab

3. Run dataset generator:

python dataset_generator.py

4. Train the model:

python train_model.py

5. Start the backend server:

python app.py

6. Open the website:

website/index.html
